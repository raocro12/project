Проект представляет из себя Spring Boot Web MVC приложение для управления библиотекой. Система позволяет работать с книгами, читателями и выдачами книг.
# Функциональность
- **CRUD для книг**: добавление, редактирование, удаление, просмотр  
- **CRUD для читателей**: добавление, редактирование, удаление, просмотр  
- **CRUD для выдач книг**: оформление выдачи, возврат, редактирование, удаление  
- Валидация данных на уровне сущностей (например, проверка формата ФИО, телефона, email, даты рождения, года публикации)  
- Проверка уникальности для ключевых полей:  
  - `Book`: комбинация `author + name + publishingHouse`  
  - `Reader`: комбинация `firstName + lastName + dateOfBirth`  
  - `BookLending`: комбинация `reader + book + dateOfIssue`
